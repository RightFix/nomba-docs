Version history

0.1.0

Initial release.

  • All 64 endpoints from Nomba's OpenAPI spec, generated across 10 resource groups.
  • Sync (Nomba) and async (AsyncNomba) clients.
  • Typed responses via generated TypedDict models.
  • Cursor pagination helpers (paginate / apaginate).
  • Guided card-payment OTP/3DS flow (CardPaymentFlow).
  • Webhook signature verification with replay-window protection.
  • Token-fetch locking and retry-with-backoff on 429/5xx.
  • Local nested request-body validation against the bundled spec.
  • Bounded-concurrency helper for async fan-out calls.
